# Programa/Código em python de Monitoramento de produção de funcionários na fábrica
O programa feito tem a função de monitorar a produção dos funcionários de uma fábrica durante 16 horas que são registradas em valores numéricos.

Além do programa guardar horários ele também pode calcular a produção total, fazer a média de produção por hora, identificar os horários mais produtivos e menos produtivos, também podendo detectar quedas na produção.
